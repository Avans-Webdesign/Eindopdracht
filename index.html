Welkom op onze super coole site!

Dit is onze eindopdracht.

Webdesign

Technisch
  • Er wordt gebruik gemaakt van HTML5, CSS3 en JavaScript (jQuery3)
  • Site is responsive via Bootstrap
    • De site is uniek op tablets en telefoons
  • HTML tags worden logisch gebruikt
  • Meerdere malen gebruik maken van :hover
  • CSS staat in een apart bestand
  • JavaScript staat in een apart bestand
  • Geen gebruik gemaakt van Wordpress, Joomla! of Drupal
  • HTML, CSS en JS is duidelijk en springt goed in
Vormgeving
  • De website ziet er professioneel, aantrekkelijk en mooi uit
  • De website heeft een unieke eigen stijl
  • Er komen symmetrische en asymmetrische elementen voor
  • Er is vooraf een prototype van de layout en stijl gemaakt
  • Er wordt niet kant-en-klare template voor de website gebruikt
  • Eventuele afbeeldingen bevatten een bronvermelding
Structuur

Al deze punten moeten in de Contact pagina voorkomen

  • Er wordt duidelijk gemaakt wie aan de site heeft gewerkt
    • Naam
    • Studentnummer
    • Email adress
    • Groepsnummer
  • Voor welke eindopdracht de site gemaakt is
  • Dat de opdracht gemaakt is in het kader van de opleiding informatica

Studievaardigheden

Inhoudelijk
  • De website heeft een titel
  • Namen en studentennummers zijn duidelijk weergegeven
  • De site is in correct Nederlands geschreven
  • Website tekst
    • Inleiding
    • Middenstuk
    • Conclusie
    • Logische alinea indeling
Lucas
  • Schrijft over persoonlijke leerdoelen
  • Ten minste vier actiepunten die gerelateerd zijn aan de studie
  • Gebruik van SMART
  • Voor elke situatie waarop gereflecteerd wordt is een beschrijving van de situatie aanwezig (bv STAR)
  • Voor elke situatie staan gevoelens en gedachten beschreven, wat er goed ging en wat er beter kan
  • Er wordt duidelijk een verband gelegd tussen gedrag en resultaat
  • De reflectie is gekoppeld aan wat er in de training is geleerd
  • Op basis van de reflectie geef je een concreet verbeterpunt aan voor de toekomst
