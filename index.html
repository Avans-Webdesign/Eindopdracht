Lucas de Jong

Hoi, ik ben Lucas de Jong en ik studeer aan Avans Hogeschool Breda met de opleiding informatica. Ik woon in de grootse Drechtstad, Dordrecht Elke schooldag reis ik naar Breda toe met de trein, wat ongeveer 20 minuutjes duurt. Helaas is de school echter niet vlakbij het station, waardoor ik vanaf Breda Centraal Station ook nog een bus richten het Avans gebouw nemen moet. Ik heb informatica altijd al leuk gevonden en ben er al van jongs af aan veel bezig geweest met computers. Dit is ook waarom ik voor een opleiding informatica gekozen heb, ik hoop een hobby in een baan te kunnen veranderen. Thuis ben ik erg veel bezig met de C# taal. C# komt op meerdere punten deels overheen met Java, de taal die wij nu geleerd krijgen. Hierdoor kan ik de programmeer lessen gemakkelijk volgen en heb ik niet veel intrinsieke motivatie om huiswerk te maken. Wanneer ik de opleiding voltooi hoop ik een leuke baan in het interessante informaticaveld te vinden.

Miranda Verheij

Webdesign

Technisch
  • Er wordt gebruik gemaakt van HTML5, CSS3 en JavaScript (jQuery3)
  • Site is responsive via Bootstrap
    • De site is uniek op tablets en telefoons
  • HTML tags worden logisch gebruikt
  • Meerdere malen gebruik maken van :hover
  • CSS staat in een apart bestand
  • JavaScript staat in een apart bestand
  • Geen gebruik gemaakt van Wordpress, Joomla! of Drupal
  • HTML, CSS en JS is duidelijk en springt goed in
Vormgeving
  • De website ziet er professioneel, aantrekkelijk en mooi uit
  • De website heeft een unieke eigen stijl
  • Er komen symmetrische en asymmetrische elementen voor
  • Er is vooraf een prototype van de layout en stijl gemaakt
  • Er wordt niet kant-en-klare template voor de website gebruikt
  • Eventuele afbeeldingen bevatten een bronvermelding
Structuur

Al deze punten moeten in de Contact pagina voorkomen

  • Er wordt duidelijk gemaakt wie aan de site heeft gewerkt
    • Naam
    • Studentnummer
    • Email adress
    • Groepsnummer
  • Voor welke eindopdracht de site gemaakt is
  • Dat de opdracht gemaakt is in het kader van de opleiding informatica
